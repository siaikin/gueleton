本示例展示了如何通过添加 `data-gueleton-bone` 属性来控制一个元素被检测为 `bone`。

示例中，我们手动将推文卡片的用户名和内容部分的父元素加上 `data-gueleton-bone` 属性，这样 Gueleton 就会将这些部分识别为一个 `bone`。
