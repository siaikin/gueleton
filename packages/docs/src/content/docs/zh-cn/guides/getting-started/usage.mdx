要让 Gueleton 正常工作，必须传递以下三个参数：

1.  `data-key`：唯一标识数据源的字符串键。
2.  `data`：数据源对象。
3.  `loading`：布尔值，指示是否显示骨架屏。

此外，Gueleton 会在默认插槽中提供一个名为 `data` 的作用域变量，代表**预存数据**或 原始的 `data`。你需要使用这个变量来渲染你的内容。就像这样。

import BasicVueRaw from '@snippets/examples/vue/Basic.vue?raw';

<FrameworkSlots>
  <Code slot="vue" code={BasicVueRaw.split('// ---cut-after---')[0]} lang="vue" meta="twoslash wrap" title="Basic.vue" />
</FrameworkSlots>

完成后，第一次渲染你的页面时，从远程获取的 `data` 和 `data-key` 会被存储在 `.gueleton/index.json` 文件中。

后续页面加载时，Gueleton 会优先使用 `.gueleton/index.json` 中的 **预存数据** 来渲染默认插槽内容，然后再根据 DOM 元素生成骨架屏。
