你可以通过 `skeleton.bone` 和 `skeleton.container` 属性传递 style 或 class 分别定制 `bone` 和 `container` 的样式。

在本文档中，Gueleton 使用 [Tailwind CSS](https://tailwindcss.com/docs/animation) 的 `animate-*` 实用类为占位骨架增加动画效果，当然你可以根据自己的需求进行修改。

## 自定义 CSS 样式

对于不使用 Tailwind CSS 的项目，文档提供了几种常见的 CSS 动画效果，你可以直接使用这些 CSS 代码，或者根据你的需求进行修改。
